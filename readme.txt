A WordPress plugin that browse content by custom fields or other attributes

== Description ==

Browse content allows to navitage throuth Wordpress content by custom fields or other attributes. 

== Installation ==

= Prerequisite = 

Install and activate Solr for WordPress plugin (http://wordpress.org/extend/plugins/solr-for-wordpress/).
Index your blog content into Solr (see Solr hosting provider here : http://www.mysolrserver.com/) 

= Installation =

1. Upload the `browse-content-by-my-solr-server` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Widget settins in configure the widget

== Frequently Asked Questions ==

= What version of WordPress does My Solr Server plugin work with? =

My Solr Server plugin works with WordPress 3.0.0 and greater.
